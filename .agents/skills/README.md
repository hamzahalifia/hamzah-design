# Workspace Agent Skills Directory

Folder ini digunakan untuk menyimpan **Agent Skills** khusus proyek ini.

## Cara Menambahkan Skill Baru:
1. Buat folder baru di dalam `skills/` dengan nama skill Anda (misal: `my-custom-skill`).
2. Buat file `SKILL.md` di dalam folder tersebut.
3. Struktur `SKILL.md`:
```markdown
---
name: my-custom-skill
description: Deskripsi singkat kapan skill ini harus digunakan oleh agent.
---

# Instruction
Instruksi detail mengenai alur kerja atau aturan yang harus diikuti oleh agent.
```
