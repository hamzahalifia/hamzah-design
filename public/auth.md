# auth.md

This document provides instructions for AI agents to register and authenticate with our services.

## Registration

To register your agent, please visit: YOUR_AGENT_REGISTRATION_URI

## Supported Identity Types

- **Identity Assertion:** We support identity assertions based on `urn:ietf:params:oauth:token-type:id-jag` and verified email.
- **Anonymous:** Anonymous access may be available for certain resources.

## Credential Types

We support standard OAuth 2.0 credential types.

## Claim and Revocation

- **Claim URI:** YOUR_CLAIM_URI
- **Revocation URI:** YOUR_REVOCATION_URI (if applicable)

Please refer to our API documentation for more details on specific endpoints and scopes.
